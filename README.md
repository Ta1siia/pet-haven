# 🐾 Pet Haven

A responsive web application that helps rescued animals find a loving home.

## 🌐 Live Demo

https://ta1siia.github.io/pet-haven/

## 📌 About the Project

Pet Haven is a team project developed during the GoIT JavaScript module.

The application allows users to:

- browse animals available for adoption;
- filter pets by category;
- view detailed information about each animal;
- submit an adoption request;
- read success stories;
- explore information about the shelter.

The project communicates with a REST API and adapts to mobile, tablet, and
desktop devices.

## ✨ Features

- Responsive layout
- Category filtering
- Animal details modal
- Adoption request form
- FAQ accordion
- Success Stories slider
- API integration
- Load More (mobile)
- Pagination (tablet & desktop)
- Toast notifications

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Vite
- Axios
- Swiper.js
- iziToast

## 🚀 Usage

Clone the repository

```bash
git clone https://github.com/Ta1siia/pet-haven.git
```

Go to the project folder

```bash
cd pet-haven
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
